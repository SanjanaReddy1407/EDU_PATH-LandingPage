# EDU_PATH-LandingPage
The EduPath is for a data-driven counseling platform that helps aspirants make confident college choices. It solves the problem of scattered, high-cost guidance. Its value proposition is providing personalized, affordable, and data-driven chat counseling to optimize students' preference lists based on their percentile.

## Features
This landing page is designed to maximize conversion by focusing on the core problem and delivering a highly polished user experience:

Responsive Design: Looks great on desktops, tablets, and mobile devices.

Video Background: Uses a full-width video loop in the Hero section for visual impact.

Aesthetic Animations: Features smooth, staggered 'Fade-In Up' animations using the Intersection Observer API for sections like Problem Cards and Feature Blocks.

Clear Call-to-Action (CTA): Guides students toward the 'Calculate Your Admission Probability' lead form.

Local Trust Signals: Highlights data coverage from key educational hubs in Maharashtra (Mumbai, Pune, Nagpur, etc.) to build immediate relevance.
